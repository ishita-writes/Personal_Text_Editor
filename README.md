A clean, responsive personal text editor designed for seamless code editing and text manipulation. This project was built to explore lightweight software architecture and custom interface styling.

Features

* **Fluid Interface:** A minimalist, focused writing environment designed to eliminate distractions.
* **Responsive Layout:** Optimized for various screen dimensions with adaptive typography.
* **File Management:** Built-in capabilities to handle, edit, and manipulate text files directly.

Built With
* **HTML5 & CSS3** - For the modern, responsive interface layout.
* **JavaScript** - For client-side state management and text input parsing.
